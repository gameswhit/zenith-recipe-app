FAQ

Common questions

How does ZenChef differ from existing tools?

ZenChef adds an AI designer (describe in English to executable workflow), MiMo-powered optimization, custom code nodes with full dependency support, and sub-second execution with warm pools.

What happens if a step fails?

Configurable retries with exponential backoff, dead-letter queues, and human-in-the-loop approval gates. Every failure is logged with full context for debugging.

Can I use my own code in workflows?

Yes. Custom code nodes support Python, JavaScript, and shell scripts with full npm/pip dependency resolution in isolated sandboxes.

Which models does it use?

MiMo Pro (default for planning), Claude Sonnet (long-form reasoning), DeepSeek V3 (math/code). Bring your own model via OpenAI-compatible endpoint.

Is my data secure?

All data encrypted in transit (TLS 1.3) and at rest (AES-256). Per-workflow secret vault. SOC 2 Type II certified. Multi-region deployment for data residency.
